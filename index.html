    About

    What is WeViCheck?

    WeViCheck enables next level UI checking with more tests to be carried out using a hybrid approach involving computer vision, optical character recognition and algorithms. Visual testing is made easy for you by an automated approach on testing for defects in more than five identified areas of interests. Stay tuned in for many more upcoming new tests.

    Tests

    Packed with tests

    • Layout

      Check the layout of UI elements and identify the available defects

    • Spelling & Grammar

      Identify spelling and grammar mistakes in the content of your UI

    • Alignment

      Identify elements that are not aligned in the UI

    • Contrast

      Identify the low contrast texts affecting UX
